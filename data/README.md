# Data Directory

This directory contains raw and processed data for CityPulse Montréal 2035.

## Structure

```
data/
├── raw/              # Downloaded datasets go here
│   ├── air_quality.csv
│   ├── heat_islands.geojson
│   ├── noise.csv
│   ├── traffic_segments.geojson
│   ├── travel_times.csv
│   ├── trees.csv
│   ├── planting_sites.csv
│   ├── canopy.geojson
│   └── vulnerability/  (unzipped files)
└── processed/        # Generated by ETL pipeline
    ├── montreal_grid.geojson
    ├── citypulse_grid.geojson
    ├── trees.geojson
    └── planting_sites.geojson
```

## Download Instructions

### Files to Download Manually

Visit each dataset page and download the resource files:

1. **Air Quality (RSQA)** → `raw/air_quality.csv`

   - URL: https://donnees.montreal.ca/dataset/rsqa-polluants-gazeux
   - Download the CSV resource

2. **Heat Islands** → `raw/heat_islands.geojson`

   - URL: https://donnees.montreal.ca/dataset/ilots-de-chaleur
   - Download GeoJSON or SHP resource

3. **Noise Measurements** → `raw/noise.csv`

   - URL: https://donnees.montreal.ca/dataset/mesures-niveaux-acoustiques
   - Download CSV resource

4. **Traffic Segments** → `raw/traffic_segments.geojson`

   - URL: https://donnees.montreal.ca/dataset/segments-routiers-de-collecte-des-temps-de-parcours
   - Download geometry resource (GeoJSON or SHP)

5. **Travel Times** → `raw/travel_times.csv`

   - URL: https://donnees.montreal.ca/dataset/temps-de-parcours-sur-des-segments-routiers-historique
   - Download CSV resource

6. **Trees** → `raw/trees.csv`

   - URL: https://donnees.montreal.ca/dataset/arbres
   - Download CSV resource

7. **Planting Sites** → `raw/planting_sites.csv`

   - URL: https://donnees.montreal.ca/dataset/emplacements-arbres
   - Download CSV resource

8. **Canopy** → `raw/canopy.geojson`

   - URL: https://donnees.montreal.ca/dataset/canopee
   - Download GeoJSON or SHP resource

9. **Vulnerability** → `raw/vulnerability.zip`
   - URL: https://donnees.montreal.ca/dataset/vulnerabilite-changements-climatiques
   - Download ZIP, unzip to `raw/vulnerability/`

### Live APIs (No Download Needed)

These are fetched directly by the ETL scripts:

- **Entraves (traffic closures)** - Live JSON feed
- **OpenWeather** - Requires API key in .env
- **STM GTFS** - Download gtfs.zip from STM developer portal

## Notes

- The ETL pipeline will automatically use real data if files are present
- If files are missing, it will fall back to simulated data for development
- Check PRD.md for detailed information about each dataset
